As this is a demo/work-in-progress, you can only search by:

  • entering the specific nucleotide index you want to jump to, e.g.: 311
  • entering a nucleotide range you want to jump to, e.g.: 77380..94720
  • More search features will be added later, for example:
  • search by feature name(s)
  • search by properties
  • search by sequence
  • search within specific cromosomes

and possibly more to come!
