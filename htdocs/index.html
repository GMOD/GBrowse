Generic Genome Browser

Welcome to the Generic Genome Browser. Before you can start using the system, you must create a genome database and load some data. Sample data from yeast is located in the gbrowse distribution in the directory sample_data. Instructions for loading this data is in INSTALL.

If you installed using PPM, you can download the yeast features and yeast DNA from the gmod downloads page. You'll also find Worm, Fly and Human genome annotations here, although not necessarily the most up to date ones..

Once you've loaded the yeast data (and adjusted the config file as described in the INSTALL document), you can browse the yeast genome by following the link http://your.host/cgi-bin/gbrowse?source=yeast.
