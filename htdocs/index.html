Example Databases

Example in-memory databases from yeast chromosomes 1 and 2 have been installed for you. Try them at these URL:

  • http://your.host/cgi-bin/gb2/gbrowse/yeast (simple example)
  • http://your.host/cgi-bin/gb2/gbrowse/yeast_advanced (advanced example)
  • http://your.host/cgi-bin/gb2/gbrowse/yeast_renderfarm (yeast_renderfarm example)

The "yeast_renderfarm" example will only work if a local gbrowse_slave render slave daemon is running. To launch the daemon, run this command as the root user: 

% /etc/init.d/gbrowse-slave start

Alternative URLs

If you allowed the install process to adjust your Apache configuration, you can use the alias /gb2 as an alternative to /cgi-bin/gb2:
  • http://your.host/gb2/gbrowse/yeast
  • http://your.host/gb2/gbrowse/yeast_advanced
  • http://your.host/gb2/gbrowse/yeast_renderfarm
  • http://your.host/gb2/gbrowse/pop_demo (demo of embedded population allele frequency maps -- requires Template module to be installed)

Accelerated Demos

If you have FastCGI and/or ModPerl installed, you will have access to accelerated versions of gbrowse a these URLs:
  • FastCGI
    • http://your.host/fgb2/gbrowse/yeast
    • http://your.host/fgb2/gbrowse/yeast_advanced
    • http://your.host/fgb2/gbrowse/yeast_renderfarm
  • ModPerl
    • http://your.host/mgb2/gbrowse/yeast
    • http://your.host/mgb2/gbrowse/yeast_advanced
    • http://your.host/mgb2/gbrowse/yeast_renderfarm

Directory Paths

For future reference, here is where you've installed GBrowse's various components:

GBrowse documentation, stylesheets and in-memory databases.
/var/www/gbrowse2

GBrowse configuration files
/etc/gbrowse2/GBrowse.conf

Preinstalled sample database(s)
/var/lib/gbrowse2/databases

GBrowse CGI (web) scripts
  • /usr/lib/cgi-bin/gb2/gbrowse
  • /usr/lib/cgi-bin/gb2/gbrowse_img
  • /usr/lib/cgi-bin/gb2/gbrowse_details

Temporary directories for cached images and sessions.
/tmp/gbrowse2
