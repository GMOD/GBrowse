Generic Genome Browser

Welcome to the Generic Genome Browser. Before you can start using the system, you must create a genome database and load some data. Sample data from yeast is located in the gbrowse distribution in the directory sample_data. Instructions for loading this data is in INSTALL. You will most likely want to use the script bp_bulk_load_gff.pl which should have been installed when you installed BioPerl.

If you installed using PPM, you can download the yeast features and yeast DNA from the gmod downloads page. You'll also find Worm, Fly and Human genome annotations here, although not necessarily the most up to date ones. Note that you will need to uncompress the data files prior to running to load script.

Once you've loaded the yeast data (and adjusted the config file as described in the INSTALL document), you can browse the yeast genome by following the link http://your.host/cgi-bin/gbrowse?source=yeast.
